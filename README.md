# Pelatihan AI Cursor — AI-Powered Coding & Developer Productivity

> Pelatihan 3 hari dari **Multimatics** untuk meningkatkan produktivitas developer dan tim IT melalui pemanfaatan AI-powered code editor (Cursor) secara optimal, aman, dan etis. Selama 3 hari Anda akan membangun **satu aplikasi nyata** bernama DevNotes dari nol sampai live di Vercel.

---

## Mulai dari Sini

1. **Baca [`pendahuluan.md`](./pendahuluan.md)** — pengantar, prasyarat, dan checklist persiapan sebelum hari pertama.
2. **Baca [`project-brd.md`](./project-brd.md)** — spek lengkap aplikasi DevNotes yang akan Anda bangun (FR/NFR, mockup, flow).
3. Lanjut ke materi & latihan Hari 1.

---

## Tujuan Pelatihan

Setelah mengikuti pelatihan ini, Anda akan mampu:

1. Memahami konsep AI-assisted coding dan peran Cursor dalam software development.
2. Menggunakan Cursor untuk menulis dan memahami kode secara kontekstual.
3. Mempercepat debugging, refactoring, dan code review dengan bantuan AI.
4. Mengoptimalkan workflow pengembangan software menggunakan Cursor.
5. Menerapkan penggunaan AI secara aman, etis, dan sesuai kebijakan organisasi.
6. Mengirim satu aplikasi full-stack (DevNotes) end-to-end: FE statis → BE Supabase → FE Next.js ter-deploy di Vercel.

## Target Peserta

- Software Developer (Backend / Frontend / Full-Stack)
- Software Engineer & Application Developer
- DevOps Engineer
- IT Development Team & Technical Lead
- Data Engineer / Automation Engineer

## Prasyarat Singkat

Detail lengkap di [`pendahuluan.md`](./pendahuluan.md#2-prasyarat).

- Terbiasa coding di minimal satu bahasa pemrograman.
- Familiar dengan Git basic dan terminal/CLI.
- Laptop ≥ 8 GB RAM, koneksi internet stabil.
- Sudah mengisi **[Pretest](pretest/PRETEST-AI-Cursor.md)** sebelum hari pertama.
