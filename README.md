# Pelatihan AI Cursor — AI-Powered Coding & Developer Productivity

> Pelatihan 3 hari dari **Multimatics** untuk meningkatkan produktivitas developer dan tim IT melalui pemanfaatan AI-powered code editor (Cursor) secara optimal, aman, dan etis.

---

## Tujuan Pelatihan

Setelah mengikuti pelatihan ini, peserta mampu:

1. Memahami konsep AI-assisted coding dan peran Cursor dalam software development.
2. Menggunakan Cursor untuk menulis dan memahami kode secara kontekstual.
3. Mempercepat debugging, refactoring, dan code review dengan bantuan AI.
4. Mengoptimalkan workflow pengembangan software menggunakan Cursor.
5. Menerapkan penggunaan AI secara aman, etis, dan sesuai kebijakan organisasi.

## Target Peserta

- Software Developer (Backend / Frontend / Full-Stack)
- Software Engineer & Application Developer
- DevOps Engineer
- IT Development Team & Technical Lead
- Data Engineer / Automation Engineer

## Prasyarat

- Sudah terbiasa coding pada minimal satu bahasa pemrograman.
- Memahami dasar Git (clone, branch, commit).
- Laptop dengan minimal 8 GB RAM (16 GB direkomendasikan).
- Koneksi internet stabil (Cursor memerlukan akses online).
- Sudah mengisi **[Pretest](pretest/PRETEST-AI-Cursor.md)** sebelum hari pertama.
